# React

Набор учебных проектов, созданных в рамках курса по React. Каждый проект демонстрирует различные аспекты разработки на React.

## Список проектов
1. Product Catalog
Каталог товаров с функциями электронного магазина.

2. Quiz App
Полнофункциональная платформа для создания и прохождения тестов с административной панелью.

3. To-Do List
Простое приложение для управления задачами с возможностью добавления, удаления и отметки выполненных задач.

# ДЕТАЛЬНОЕ ОПИСАНИЕ QUIZ APP

## Ключевые возможности

**Для пользователей:**
- **3 типа вопросов:**

- **Создание вопросов** всех трех типов
- Управление вариантами ответов

# Установка и запуск

Следуйте этим шагам, чтобы запустить проект на вашем компьютере.

## Рекомендуемые требования:

- Node.js: 18.0.0+ LTS версия
- npm: 9.0.0+ или yarn: 1.22.0+

## Пошаговая инструкция

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Andreylab/react.git
    ```

2.  **Перейдите в директорию проекта:**
    ```bash
    cd react
    ```

3.  **Установите зависимости:**
    ```bash
    npm install
    ```
    Эта команда загрузит все необходимые библиотеки, указанные в `package.json`.

4.  **Запустите приложение в режиме разработки:**
    ```bash
    npm run dev
    ```
    
5.  **Откройте браузер:**
    Приложение откроется по адресу [http://localhost:5173](http://localhost:5173). Страница будет перезагружаться при внесении изменений в код.

## Описание структуры

**Компоненты (components/)**
- **Blocks/** - Крупные компоненты-секции для построения страниц
- **QuizCreation/** - Специализированные компоненты для создания тестов и квизов
- **UI компоненты** - Переиспользуемые элементы интерфейса (Button, Counter, Header)
- **Функциональные компоненты** - Компоненты с конкретной логикой (Todo, Exchange, Post)

**Страницы (pages/)**
- **QuizesPage/** - Все связанное с отображением и прохождением квизов
- **Административные страницы** - Управление контентом и пользователями
- **Пользовательские страницы** - Основной функционал для конечных пользователей
- **Layout страницы** - Общие макеты приложения

**Системные файлы**
- **App.jsx** - Корневой компонент, точка входа в приложение
- **routes.js** - Конфигурация маршрутизации между страницами
- **stores.js** - Управление глобальным состоянием приложения
- **package.json** - Метаданные проекта и зависимости

## Особенности архитектуры

- **Модульность** - Компоненты сгруппированы по функциональности
- **Масштабируемость** - Легко добавлять новые компоненты и страницы
- **Переиспользуемость** - UI компоненты можно использовать в разных частях приложения
- **Разделение ответственности** - Четкое разделение на компоненты, страницы и логику
